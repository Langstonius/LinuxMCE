Summary of security features

• Unlimited number of zones and sensors
• Works with all types of sensors: motion, door/window, smoke, air quality, temperature, severe weather, etc.
• Compatible with many alarm panels see list, and with existing monitored alarm service
• 6 modes: at home (disarmed), at home secure (partially armed), armed (normal away), sleeping, extended away (you will be gone overnight), entertaining (you are expecting guests)
• Speech synthesis (ie verbal announcements, countdown timers, etc.)
• Specify how each sensor should behave in each mode: trigger a security breach, sound a warning in the house, or make an announcement (entertaining mode, for example, can announce when people come through the door so you can greet your guests)
• Custom event handling tied to sensors (ie turn on this light when this sensor is tripped)
• Unlimited number of security cameras (IP based and analog)
• Video archive accessible via the internet
• Motion detection algorithm for video cameras
• Easy association of sensors/doors/intercoms to surveillance cameras so the correct video always appears
• In response to an event, the user can specify: notify one or family members on their Pluto-enabled mobile phone, notify one or more external phone numbers, or combinations
• Live video viewing over the internet or mobile phone
• Ability to speak to any or all of the house remotely
• Passive, or automatic, arming and disarming
• Single button arming with the mobile phone
• "Follow-me" security, allowing certain authorized users to enter certain rooms with automatic bypass of those sensors how does it work
• Unlimited event log
• User-definable event handlers for each sensor
• Countdowns, messages, and status displayed on the Orbiters
