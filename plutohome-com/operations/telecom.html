Summary of Telecommunications Features

• Integrated digital pbx phone switch
• Up to 1,000 extensions and 100 incoming lines
• Handles POTS (normal analog phone), ISDN, T1 and voice-over-ip
• Unlimited number of simultaneous calls with voice-over-ip
• Unlimited number of worldwide and toll-free phone numbers using voice-over-ip
• Unlimited number of voicemail boxes
• Voicemail accessible using any phone, or over the internet, or by email, or playback through the tv/stereo
• Auto attendant with call routing
• System flags certain incoming calls (by caller id) as being for certain persons for auto routing
• User designate certain incoming lines for auto-routing to certain users
• Each user has 4 status settings or modes: at home, away, sleeping and do not disturb, with custom call routing for each mode
• User can designate priority callers who receive special handling (such as the ability to wake the user if his mode is sleeping)
• Multiple homes joined together as one using voice-over-ip. Any call to any home can ring in any other Pluto-equipped home
• Individual speed dial and recent call list for each family member
• "Super-simple" transfer/conference and other features
• Pre-recorded professional voice prompts, or record your own
