Summary of Personal Computing Features

The Pluto Core:

• Linux-based server, dual Intel Xeon processor, up to 10TB RAID 5 array, dual NIC adapters (internal and external LAN's) with Gigabit (1,000 BaseT) Ethernet
• 4U 19" rackmount
• Integrated web server (Apache), email server (IMAP and POP3), customized DHCP server for self-configuring and diskless boot
• Zero-configuration system

The Pluto Media Director:

• Runs both Linux and Windows
• Linux system is diskless-network boot--all files stores on the Core
• Linux system includes popular software: OpenOffice, Mozilla web browser, etc.
• Wiresless keyboard+mouse
• Gigabit Ethernet, firewire, USB2, digital film slots
• CD/DVD -R/RW and +R/RW drive
• Hardware-based MPEG2 encoding for PVR
